# lang-french-extended

## Installation

Use [Bazaar](https://github.com/flagrow/bazaar/) or install manually via Composer:

```bash
composer require rooaaar/lang-french-extended
```

## Updating

```bash
composer update rooaaar/lang-french-extended
```
